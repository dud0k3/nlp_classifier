# NLP-проект по архиву Twitter Elections Integrity

Полноценный NLP-проект на Python, построенный на основе архивированной статической версии сайта Twitter, связанного с темой **Elections Integrity**.

## О проекте

Этот проект показывает полный NLP-пайплайн на реальных веб-документах:

- извлечение текста из HTML-страниц;
- очистка и нормализация текста;
- формирование корпуса документов;
- построение TF-IDF-представления;
- поиск похожих документов по текстовому запросу;
- извлечение ключевых слов;
- кластеризация документов.

Проект подходит как учебная и портфолио-работа для GitHub, потому что показывает не только модель, но и полный путь от сырых данных до работающего анализа.

---

## Особенность датасета

Важно: архив `twitter-data-staticsite.zip` — это не классический ML/NLP-датасет с готовыми метками классов и не архив с обученной моделью.

Внутри находятся:

- HTML-страницы;
- стили и скрипты;
- изображения и другие статические файлы.

Поэтому задача проекта построена корректно:  
сначала мы извлекаем текст из страниц, затем применяем к нему NLP-методы.

---

## Что умеет проект

### 1. Извлечение документов из HTML
Скрипт проходит по архиву, находит HTML-файлы, очищает их от служебных тегов и сохраняет текст в таблицу документов.

### 2. Построение NLP-индекса
Для корпуса документов создаются:
- TF-IDF-векторы;
- словарь признаков;
- кластеры документов через `KMeans`.

### 3. Поиск по документам
Можно ввести текстовый запрос и получить наиболее релевантные страницы.

### 4. Извлечение ключевых слов
Для каждого документа можно получить наиболее важные слова и словосочетания.
